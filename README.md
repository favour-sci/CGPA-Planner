## Student Grade Planner Pro

The Student Grade Planner Pro is a powerful, interactive, and privacy-focused dashboard designed to help students track, predict, and visualize their academic performance. It allows users to input test and exam scores to forecast their GPA/CGPA dynamically. Features include automatic data visualization, grade scale toggling (4.0/5.0), and data integration. Built with **HTML5**, **CSS3**, and **JavaScript**, utilizing **Chart.js** for real-time analytics.

The **Student Grade Planner Pro** solves the problem of manual CGPA calculation and lack of academic visualization. It moves beyond simple spreadsheets by providing a "What-If" analysis toolâ€”allowing students to "paint their scenario" by inputting potential scores to see how they impact their final standing.

Because it runs entirely in the browser using **Local Storage**, student data remains private on their device and persists between sessions.

## Key Features

* **Interactive Dashboard:** Real-time visual feedback using Gauges for GPA/CGPA, Bar Charts for top courses, and Donut Charts for grade distribution.
* **Flexible Grading Scales:** Instantly toggle between **5.0 Scale** and **4.0 Scale**.
* **Auto-Save Functionality:** Never lose your data! Inputs are saved automatically to the browser's Local Storage.
* **"Paint Your Scenario":** Input Test and Exam scores to calculate Weighted Grade Points (WGP) dynamically.
* **Easy Management:** Add or delete courses with a single click.
* **Fully Responsive:** Works perfectly on desktops, tablets, and mobile phones.
* **Privacy First:** No database required. All data stays on the user's device.

## Technologies Used

* **Core:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Visualization:** [Chart.js](https://www.chartjs.org/)
* **Plugins:** [chartjs-plugin-datalabels](https://chartjs-plugin-datalabels.netlify.app/) (for data values on charts)
* **Icons:** [FontAwesome 6](https://fontawesome.com/)

## How to Use

1.  **Select Your Scale:** Use the dropdown in the header to choose between a 5.0 or 4.0 grading system.
2.  **Input History (Optional):** If you are a returning student, enter your **Previous CGPA** and **Previous Total Units** in the summary panel.
3.  **Add Courses:** Click "Add Course" to create rows for your current semester.
4.  **Enter Scores:** Type your Course Code, Units, Test Scores, and Exam Scores. The system auto-calculates the Total, Grade, and Points.
5.  **Analyze:** Watch the dashboard at the top update instantly to reflect your projected GPA and Class of Degree (e.g., First Class, Second Class Upper).

## Installation & Setup

Since this is a client-side application, you do not need to install Node.js or set up a database.
